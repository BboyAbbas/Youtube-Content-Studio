=========================
USAGE NOTES:
=========================

1. ALWAYS use the exact description when a character appears
2. Main character NEVER gets a description beyond "cartoon person"
3. If a new character type is needed, add them to this sheet first
4. Keep descriptions consistent across ALL scenes featuring that character
5. End all prompts with "Simple cartoon style with clean lines."

EXAMPLE USAGE:

Bad: "A soldier giving advice"
Good: "A soldier wearing camouflage fatigues, combat boots, dog tags around neck, determined expression, giving advice to a standing cartoon person"

Bad: "The person's mom talking to them"
Good: "Mom figure wearing casual blouse, apron, hair in a bun, hands on hips, concerned motherly expression, talking to a standing cartoon person"
